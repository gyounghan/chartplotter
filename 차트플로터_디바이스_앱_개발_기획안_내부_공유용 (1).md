# 차트플로터 디바이스 & 앱 개발 기획안 (1차)

---


## 1. 추진 배경 및 목적

### 1.1 추진 배경

- **상용 차트플로터의 한계**: 기존 상용 차트플로터는 고가이며, 폐쇄적 구조로 **확장과 차별화에 한계**가 있음. 커스터마이징 및 확장에 제약이 큼
- **현장 요구 증가**: AIS, 카메라 등 사용자 요구는 증가하고 있으나 유연한 대응 부족. 대응이 느림
- **경쟁 환경**: 타사 제품(킹피쉬 등)이 이미 시장에 출시되어 판매 중이나, **당사만의 현장 경험과 소프트웨어 역량을 결합한 차별화된 솔루션** 제공 가능
- **당사의 강점**: 항해·통신장비 **유통·설치·현장 경험 + 소프트웨어 역량**을 동시에 보유. 소프트웨어 기반 확장을 통해 차별화 가능

→ **자체 디바이스 + 앱 기반 플랫폼**으로 차별화 기회 확보

### 1.2 개발 목적

**우리가 만들고자 하는 것**

- **전자해도 기반 차트플로터 앱**을 통한 통합 항해 플랫폼 구축
- **AIS 신호 수신 및 전용 앱 구현**
  - 국내, 해외, 선박용도에 따라 선 설정
  - 지인 배, 특정 선박 MMSI 등록 가능 (색상 구분 표시)
  - 타겟 선박의 항적 기능
  - 접근/인접 시 경고 기능
- **안드로이드 디바이스(플로터)와 외부 카메라 연결**하여 화면에 표시 + 블랙박스 기능
- **안드로이드 디바이스 내 런처앱 개발** - 선박용으로 최적화
- 레저선, 연안 어선 등 다양한 고객군 대응

### 1.3 기대 효과

#### 1. 비용·유연성 경쟁력 확보
- **하드웨어 비용 절감**: 안드로이드 디바이스 기반으로 전용 하드웨어 개발 비용 절감
- **소프트웨어 중심 구조**: 앱 기반으로 기능 추가/수정이 빠르고 유연함
- **확장성**: 외부 카메라, AIS 등 추가 장비 연동

#### 2. 디바이스 판매 + 앱·서비스 확장 가능성
- **하드웨어 판매 수익**: 안드로이드 디바이스 + 선박용 케이스/마운트 패키지 판매
- **장기 수익 구조**: 1회 하드웨어 판매 후 지속적인 소프트웨어/서비스 개선으로 인한 고객 확보

#### 3. 다양한 고객군 대응
- **레저선**: 기본 항해 기능 중심
- **연안 어선**: 어장 정보, 항적 기록 강화, AIS 타겟 관리 집중
- **시장 확장성**: 각 고객군별 특화 기능을 모듈화하여 추가 판매 가능

## 2. 전체 시스템 구성 개요

### 2.1 시스템 구성 개념

- **안드로이드 디바이스(플로터)**: 전자해도 차트플로터 앱 실행, 항해 중 핵심 정보 표시 및 조작
- **AIS 전용 앱**: AIS 신호 수신, 선박 추적, 타겟 관리, 경고 기능
- **외부 카메라**: 화면 표시 및 블랙박스 기능
- **선박용 런처앱**: 안드로이드 디바이스 부팅 시 선박용 환경 제공
- 서버: 설정/데이터 동기화, 업데이트, 확장 서비스 기반

```
[GPS / AIS 수신기 / 외부 카메라]
          ↓
[안드로이드 디바이스 (플로터)]
  ├─ 전자해도 차트플로터 앱
  ├─ AIS 전용 앱
  ├─ 카메라 연동 앱
  └─ 선박용 런처앱
          ↓
       [서버]
```

### 2.2 역할 분리 원칙

- **안드로이드 디바이스**: 실시간성, 안정성 최우선, 선박 환경 최적화
- **차트플로터 앱**: 전자해도 표시 및 항해 정보 관리
- **AIS 앱**: 선박 추적 및 경고 기능 전담
- **카메라 앱**: 영상 표시 및 블랙박스 기능
- **런처앱**: 선박용 환경 제공 및 앱 간 전환
- 서버: 데이터 축적 및 장기 확장 기반

## 4. 앱 기능 정리

### 4.1 전자해도 차트플로터 앱

#### 역할 정의
- 안드로이드 디바이스에서 실행되는 **메인 차트플로터 앱**
- 전자해도 표시 및 항해 정보 관리

#### 주요 기능
- 전자해도 기반 차트 표시
- GPS 위치, 헤딩, COG, SOG 정보 표시
- 항적 표시 및 저장
- 포인트 관리 (등록, 수정, 삭제)
- 목적지 설정 및 항해 경로 표시
- 낚시금지구역 경고 알림
- **지도 표시 옵션**
  - 2D / 3D 지도 모드 및 동기화
  - 3D 보트 아이콘 설정 (세일보트 / 배스보트 / 콘솔중앙)
  - 거리원(가늠원) 표시
  - 헤딩선 연장 표시
  - GPS 코스선 연장 표시
  - 위경도 격자선 표시
- **레이어 표시 제어**
  - 목적지 표시 ON/OFF
  - 경로 표시 ON/OFF
  - 항적 표시 ON/OFF
  - 지도 표시 ON/OFF
- **지도 조작 기능**
  - 커서 복귀 (현재 위치로 이동)
  - 측정 기능 (거리/면적 측정)
  - 찾기 기능 (위치 검색)
  - 팝업 정보 표시 (객체 선택 시)
- **회전 방향 설정**
  - North Up (노스업)
  - Heading Up (헤딩업)
  - Course Up (코스업)
- **저장소 관리**
  - 목적지 / 경로 / 항적 저장소 이동 및 관리

 │   │   ├─ 3D 보트 선택

 │   │   │      (세일보트 / 배스보트 / 콘솔중앙)
 │   │   │

 │   │   │
 │   │   ├─ 코스 연장
 │   │   │   └─ GPS선 ON / OFF
 │   │   │
 │   │   ├─ 연장 길이
 │   │   │   └─ 연장선 길이 설정
 │   │   │
 │   │   ├─ 2D / 3D 동조
 │   │   │   └─ (2D↔3D 지도 동기화)
 │   │   │
 │   │   ├─ 팝업 정보

 │   │   ├─ 위경도선

 │   │   │

 │   │   │   └─ 목적지 표시 ON / OFF
 │   │   │
 │   │   ├─ 경로

 │   │   │

### 4.2 AIS 전용 앱

#### 역할 정의
- AIS 신호 수신 및 선박 추적 전용 앱
- 타겟 선박 관리 및 경고 기능 제공

#### 주요 기능
- **AIS 신호 수신 및 주변 선박 표시**
- **선박 분류 설정**
  - 국내/해외 선박 구분
  - 선박 용도별 분류 (어선, 상선, 레저선 등)
- **특정 선박 관리**
  - 지인 배, 특정 선박 MMSI 등록
  - 등록된 선박 색상 구분 표시
- **타겟 선박 항적 기능**
  - 선택한 선박의 이동 경로 추적
  - 항적 기록 및 재생
- **접근/인접 경고**
  - 등록된 선박 접근 시 알림
  - 위험 거리 내 선박 감지 시 경고

#### 주요 기능

### 4.4 선박용 런처앱

#### 역할 정의
- 안드로이드 디바이스 부팅 시 실행되는 런처앱
- 선박 환경에 최적화된 가로 모드 인터페이스 제공
- 항해 중 필요한 모든 기능에 대한 통합 접근점 제공

#### 화면 구성 (가로 모드)

런처앱은 가로 모드로 구성되며, 화면을 3개 영역으로 구분합니다:

**왼쪽 영역**: 빠른 접근 메뉴
- 설정, 목적지, 경로, 항적, 물떼, AIS정보, 탐색기 메뉴가 세로로 배치
- 각 메뉴를 클릭하면 해당 기능의 설정/관리 화면으로 이동

**중앙 메인 영역**: 앱 실행
- 전자해도, 카메라, AIS, 계기판 4개 앱을 큰 아이콘으로 2x2 그리드 형태로 배치
- 각 아이콘을 터치하면 해당 앱이 실행됨

**우측 영역**: 화면 분할 기능
- 화면 분할 기능을 활성화하여 여러 앱을 동시에 화면에 표시 가능

#### 주요 기능

**왼쪽 영역 (빠른 접근 메뉴)**
- **설정**: 시스템 전체 설정 화면 진입
- **목적지**: 목적지 관리 및 설정 화면
- **경로**: 경로 관리 및 편집 화면
- **항적**: 항적 관리 및 조회 화면
- **물떼**: 물떼(포인트) 관리 화면
- **AIS정보**: AIS 선박 정보 및 타겟 관리 화면
- **탐색기**: 파일 탐색기 (저장된 데이터 관리)

각 메뉴 클릭 시 해당 기능의 설정/관리 화면으로 이동

**중앙 메인 영역 (앱 실행)**
- **전자해도**: 차트플로터 앱 실행 (큰 아이콘)
- **카메라**: 카메라 연동 앱 실행 (큰 아이콘)
- **AIS**: AIS 전용 앱 실행 (큰 아이콘)
- **계기판**: 계기판 앱 실행 (큰 아이콘)

4개 앱을 큰 아이콘으로 표시하여 터치 조작 용이

**우측 영역 (화면 분할)**
- 화면 분할 기능 활성화
- 여러 앱을 동시에 화면에 표시 가능
- 예: 전자해도 + AIS 동시 표시, 카메라 + 계기판 동시 표시 등

#### 추가 기능
- 부팅 시 자동 실행
- 선박 환경에 최적화된 가로 모드 고정
- 빠른 앱 전환 및 멀티태스킹 지원
